# Random Pattern Generator

A simple CLI-based Random Pattern Generator built in C++. This program generates random text patterns directly in the terminal based on user input.

## Features

* Generate random patterns from the command line
* Fast and lightweight
* Easy to use
* Beginner-friendly C++ project
* No external dependencies required

## Installation

### Clone the Repository

```bash
git clone https://github.com/your-username/random-pattern-generator.git
cd random-pattern-generator
```

### Compile the Program

Using g++:

```bash
g++ main.cpp -o pattern_generator
```

### Run

```bash
./pattern_generator
```

## Usage

After running the program, follow the on-screen prompts to generate random patterns.

Example:

```text
Enter pattern size: 5

Generated Pattern:

* * * * *
  * * *
* * * * *
  * * *
* * * * *
```
